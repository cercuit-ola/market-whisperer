## 🧠 What is MarketBrief?

**MarketBrief** (formerly *market-whisperer*) is an AI-powered web application that delivers concise, intelligent daily market intelligence to investors, traders, and financial enthusiasts. Instead of spending hours parsing news feeds, earnings reports, and price charts, MarketBrief synthesizes it all into clean, actionable briefings — powered by large language models.

Whether you're a retail investor checking in before market open or a analyst looking for a high-level pulse, MarketBrief whispers the signal through the noise.

- 📰 **Daily Market Briefings** — AI-generated summaries of major market movements, trends, and events

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) v18+ or [Bun](https://bun.sh/) v1+
- An AI API key (e.g. DigitalOcean Gradient AI, OpenAI, or compatible provider)

### Installation

```bash
# Clone the repository
git clone https://github.com/cercuit-ola/market-whisperer.git
cd market-whisperer

# Install dependencies (using Bun — recommended)
bun install

# Or with npm
npm install
```

### Environment Variables

Create a `.env` file in the root of the project:

```env
# AI Provider (DigitalOcean Gradient AI recommended)
VITE_AI_API_KEY=your_api_key_here
VITE_AI_BASE_URL=https://inference.do-ai.run/v1   # or your provider's endpoint
VITE_AI_MODEL=your_model_name

# Optional: Market data source
VITE_MARKET_API_KEY=your_market_data_key
```

> ⚠️ Never commit your `.env` file. It is already listed in `.gitignore`.

### Running Locally

```bash
# Start the development server
bun run dev

# Or with npm
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Building for Production

```bash
bun run build

# Preview the production build locally
bun run preview
```

## 🌐 Deployment

MarketBrief is optimized for deployment on [Vercel](https://vercel.com/).

### Deploy with Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel --prod
```

Set your environment variables in the Vercel dashboard under **Project → Settings → Environment Variables**.

## 🏆 Hackathon: DigitalOcean Gradient™ AI Hackathon

This project was built as a submission for the **[DigitalOcean Gradient™ AI Hackathon](https://devpost.com/submit-to/28141-digitalocean-gradient-ai-hackathon)** — a global competition to build AI-powered applications on DigitalOcean's full-stack Gradient AI infrastructure.

### How MarketBrief uses DigitalOcean Gradient™ AI

- **Serverless LLM Inference** — Market summaries and AI briefings are generated via DigitalOcean Gradient AI's inference endpoint, providing scalable, low-latency AI responses without managing GPU infrastructure.
- **OpenAI-Compatible API** — The app integrates with Gradient AI's OpenAI-compatible endpoint, making it easy to swap models and scale.
- **Production Deployment** — Hosted on Vercel with Gradient AI inference called at runtime, keeping the stack lean and deployable in minutes.
